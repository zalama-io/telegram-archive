# 📦 TeleVault

> Archive Telegram Public Channels into a beautiful offline library.

TeleVault is an open-source Python tool designed to archive Telegram public channels into a fully searchable offline collection.

## ✨ Features

- Archive unlimited public Telegram channels
- Download all media
- Organize by Year → Month
- Beautiful offline HTML interface
- Works completely offline
- Resume interrupted downloads
- SQLite database
- Android (Termux) support
- PWA support
- Dark mode
- Fast search
